# TaskFlow

TaskFlow is a modern, responsive task management web app built with plain HTML, CSS, and vanilla JavaScript.

## Features

- Create, edit, complete, and delete tasks
- Search, filter, and sort tasks instantly
- Dashboard statistics with animated progress bars
- Dark/light mode with localStorage persistence
- Local storage persistence for tasks and theme
- Export and import tasks as JSON

## Run locally

Open the project folder in a browser, or serve it with a simple HTTP server.

Example:

```bash
cd TaskFlow
python -m http.server 8000
```

Then open http://localhost:8000 in your browser.
